# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2025-01-10

### Added
- Initial public release
- Documentation-first approach with comprehensive README
- Simulation-only SDK skeleton
- Core components: ControlledAgent, SimulationEngine, data schemas, MCP layer
- Behavioral constraints and safety-first design principles
- Audit trail and transparency features (placeholder implementation)
- Project structure optimized for research and educational use

### Notes
- Early-stage project with evolving APIs
- Focus on simulation environments and analytical research
- No real-world execution capabilities
- All operations are deterministic and reproducible
